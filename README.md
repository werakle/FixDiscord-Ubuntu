# FixDiscord-Ubuntu
Фикс для дискорда на Ubuntu/Linux - проверка живых стратегий и фаст подключение. memphis.sh - Проверка на доступность, dscript.sh - проверка доступных стратегий и текущего доступа.
Установка - git clone https://github.com/werakle/FixDiscord-Ubuntu/
cd FixDiscord-Ubuntu 
sudo bash memphis.sh/dscript.sh
